## 生态地形图（block 核心）—— 嵌入 AI 生成插画

地形板 `.mapplate` = 裱框容器（`border-radius:16px; overflow:hidden`），里面塞 16:9 生成 PNG。两种 mold（DNA 写死在 `assets/gen_illustration.py`）：

| mold | `--mold` | 风格 |
|------|----------|------|
| **动物森友会（默认）** | `a` | 暖萌软圆角卡通；奶油 `#f0ece2`/青 `#19c8b9`/木 `#725d42`/黄 `#ffcc00`/草绿 `#6fba2c`；小岛地形、木牌地名 |
| **pixel + cyber-hacker** | `c` | 16-bit 像素地形 + 赛博黑客；暗底 + CRT/glitch；霓虹绿青洋红；终端地名 |

`gen_illustration.py` 的 COMMON 已写死「俯瞰生态地形 + 价值之河 + 瓶颈红牌(隘口/坝) + 价值捕获金牌(宝藏堆) + 继刚作小测量员」——frame 只管「这个行业的地形怎么布」。

## 怎么把研究译成地形 frame（最关键）

deep research 出「价值链 + 瓶颈在哪环 + 价值捕获在哪环」后，把它译成一段**英文地形构图**（frame）：

- **价值之河**：价值从上游流向下游 = 一条河/路从地形一端蜿蜒到另一端。
- **各环节 = 地貌**：上游原料/设备 = 山脉/矿场；制造 = 工坊/熔炉谷；平台/渠道 = 港口/集镇；终端/用户 = 外海/城。（环节是地貌＝功能位，公司作地名小字。）
- **瓶颈 → 收窄的隘口/水坝/独木桥**（河在此被卡），插红牌。
- **价值捕获 → 金币宝藏堆/金库**（钱在此沉），插金牌。错位就让宝藏堆远离创造它的那片田；重合就让宝藏堆挨着隘口。
- **继刚** = 小测量员立在高处俯瞰（COMMON 已含，frame 可点他站哪）。
- **地名 3-6 个**、每个 2-6 字（行业关键环节名）。别更多——生图会糊。

行业结构不同，地形不同：供应链型 = 河谷（上游山→下游海）；平台型 = 中心岛 + 多边码头；分层型 = 层叠梯田/山坡。

## 出厂自检

- [ ] 瓶颈隘口🔴（红牌）与价值捕获宝藏🟡（金牌）一眼可辨、位置对？
- [ ] 价值之河流向清晰（上游→下游）？错位/重合读得出？
- [ ] 继刚测量员认得出（眼镜胡子）、不抢地形主角？
- [ ] 中文地名**正确无糊**？（糊了调 frame 减地名、重生）
- [ ] 图 = thesis/caption 讲的那张地形（同一幅）？
- [ ] base rate 三栏 + 三大问题在卡的文字块、清晰？
- [ ] 卡身强调色暖色/草绿（动森）、只点缀？
- [ ] 文本两端对齐、高度自适应、底部无留白（fullpage）？

## 踩坑清单

1. **退化成装饰风景**：地形位置要有理由（河流向/上下游/隘口卡谁/金堆哪）。说不出 = 空岛。
2. **图里别堆精确件**：地名 ≤6 个、短；base rate 数字 + 三大问题进 HTML 文字块，别进图（生图糊多标签长数字）。
3. **瓶颈/价值捕获必标清**：红牌隘口 + 金牌宝藏，一眼分得开；没标清就调 frame 重生。
4. **生图必看图**：非确定 + 偶尔糊中文 / 继刚不像——必 Read 亲验、调 frame 重生。
5. **两套色别串**：卡身强调色从动森调色板挑（暖/草绿）；地形配色是生成图自带。
6. **/tmp 文件名竞争**：并行铸卡临时 HTML + 生成图用唯一名（带行业 slug）。
7. **批量**：`gen_illustration.py` 直调 marswave、绕 listenhub 交互门控，可进批量；每张要联网+API+看图一回合。
