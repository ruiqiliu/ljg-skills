# ljg-book: 拆书

输入一本书，输出作者拿哪副取景框、答哪个问题、照出了什么。

## 拆书是什么

不是读后感，不是章节摘要，不是夸。是把作者那副取景框抽出来，看它对准了什么、照出了什么、能照多远。

一道等式撑着整件事：**看世界 = f(x)**。x 是作者站在的那个问题，f 是他看这个问题用的取景框，f(x) 是透过这副框照出的画面加他落下的那句结论。每本认真写的书，都是作者拿一副 f 去答一个 x，产出一幅 f(x)——别人也答过这个 x，有共识压着，作者觉得不对，才动笔。所以书的价值不在它说了多少，在它这副 f 把认知挪动了多少：之前大家用旧框看见 X，作者换一副框，看见 Y。

中心一问，贯穿全程：**这副 f，是作者猜想造出来的新框，还是从仓库里捡的一副现成框拿来用？** 造和捡，是两种完全不同的贡献，也拆出两张完全不同的脸。这一问，比「这本书讲了什么」尖得多。

## 脊柱：x → f → f(x)

整篇只认一条脊柱：x → f → f(x)。但脊柱长在肉里，不裸露成抽屉——它不是标题，是每一段必须干完的活。一本书该分几段、每段叫什么，跟着这本书走；脊柱只管这条线按 x、f、f(x) 的次序走完。

读者要能一眼读出这三件事。所以全篇开头，正文之前，先给一行罗盘——唯一一处把 x / f / f(x) 直接点名的地方：

```
- *x*：这本书在答的那个问题
- *f*：作者这副取景框（用这本书自己的词），后缀一词裁决：造 / 捡 / 捡+一道加工
- *f(x)*：透过这副框照出的画面 + 落成的那句话
```

三行，五秒扫完，x、f、造还是捡、f(x) 全在上面。槽位固定，填进去的全是这本书的东西——罗盘里一个套话都不许有。这一行是索引，是可靠性的锚；下面的正文，每本书长得都不一样。

## 五件活

五件活连成一条论证线，不是五个抽屉。段数随书浮动（4-6 段）：造的书可以把 f 拆成「这副框是什么」和「他怎么搭起来的」两段慢慢铺；捡的书可以把零点并进 x 一段带过。次序铁定，叫法自由——每个标题，是这本书在那一步真正干了什么的一句复述，不是诗。光扫标题，就走了一遍 x→f→f(x)。

每件活下面写的是「这一段要干的活」和「过不过得了关」。**不给现成句子。** 你看到的招式是招式，不是让你抄进正文的模板话。

### 活一 ｜ x：作者站在哪个问题前

把问题从话题里剥出来。话题是「科技与未来」，问题是「科技演化有没有自己的方向」。问题找错，后面全错。

问题不会凭空活着。它还值得写一本书，是因为它的默认答案撑不动了——作者看着大家都信的那个答案，觉得不对。所以剥出问题的同时，点名它对着哪个默认答案、那答案此刻为什么松了。这个默认答案，活二摆零点时接着用。

怎么找：读导言找「我写这本书是为了……」那一句；读不到，看作者绕了几百页都甩不掉的那一处。书名往往是答案的种子——《必然》两个字，是冲着怕科技的人说的：挡不住，焦虑没用。

关：这段拿去描述同领域另一本书，还成立吗？成立 → 太泛，重写到具体。

### 活二 ｜ 零点：这问题上已经在场的几副 f

价值是挪动，量挪动先得有零点。零点就是作者动笔前，这个 x 上已经摆着的几副取景框——主要流派各看见什么，哪一副是共识。

列 2-4 个立场，每个一行：谁（流派／人物）+ 这副框让他看见了什么。标出哪个是作者要冲的共识：教科书立场、外行常识、领域里不证自明的那套。找不到明确流派，退一步问：读者翻开这本书之前，对这个问题默认怎么想？那就是基线。

基线要正对作者的不满。摆对了，作者看见会说「对，我就是冲这个来的」。摆一堆与他无关的学术流派 = 摆错，重摆。

（薄书：这一段可以并进活一。这是「已经在场的 f 之场」，是后面量 delta 的零点，别稀释成背景介绍。）

### 活三 ｜ f：作者这副取景框，造还是捡

这是中心一段。先把 f 说清楚，再下中心裁决。

**先说框。** 用这本书自己的词，把作者这副取景框平铺直说。框不一定是名词——可以是一个公式（Shannon 的 H = -Σp log p）、一个区分（自然态 vs 文化态）、一个动作（还原到 root cause）、一个对照。同时亮出作者的独占标签：只有这本书才有、别人没有、能直接引的那些术语／列表／区分。少了独占标签，这段就能套到同领域任何一本书上——那是拆了「领域」，没拆「这本书」。

**再下裁决：造，还是捡？**

- **捡**：作者用的是一副已经有名字的现成框（行为经济学、系统论、演化论、控制论……），把它对准一个新对象。捡不丢人——很多好书捡得准。但它的贡献在「用得好」，不在「框本身」。诚实点名：从哪一派、谁那里接来的。
- **造**：作者搭出了一副仓库里没有的框，靠猜想立起来。
- **捡 + 一道加工**：捡了现成框，但加了自己的一道工序——一张表、一套人人能上手的练习、一个把框从 A 域搬到 B 域的接口。Kleon 捡了「一切皆重组」，加的是「把它做成新手每天能练的动作」。框是借的，加工是他的，分开记。

「造」「捡」「仓库」「承重件」是这份 skill 教你分辨的脚手架词，别原样写进正文。下裁决时用这本书自己的形象说——这框是借来的、是现成的、是从某派接过来的；别两本书都写「从仓库里捡」。

**怎么验是不是真造，两道：**

1. **hard-to-vary（改一处即塌）**：抽掉这副框的一根承重件，整副还转得动吗？转不动了 → 刚性，每个零件都在挣命，是搭出来的。随便换个零件还照样成立 → 松，那多半是套话或装饰，偏捡。塔勒布的凸性，抽掉「延森不等式让凸函数从波动里赚差价」这一根，整副就塌——刚的。
2. **reach（够不够远）**：这副框管不管得到它当初没被造来管的地方？这一条压到终章「走两步」里兑现，别在这里提前花掉。

**先验默认是「捡」。** hard-to-vary 过了、reach 也够，才升格成「造」。多数书是捡，或捡 + 一道加工；换轴级的真造，少。别因为「造」听着高级就到处判造——那是奉承，不是看清。

裁完，delta 自然落出来：作者这副 f 相对零点挪了什么——之前大家用旧框看见 X，作者看见 Y。挪动分三种，照实说是哪种，**但不排名次、不预设、不往「最大那档」上靠**：

- **修正**：共识大体对，作者补了一块（X 还在，多了 X'）。
- **倒转**：共识答反了，作者翻过来（X 变成非 X）。
- **换轴**：作者离开了场里那根轴，另起一根——旧问题整个作废。换轴往往就是真造：他的框在旧坐标系里压根没有位置。

（造的书：这一段是全篇的戏，hard-to-vary 那一步可以慢、可以铺开，那是一次搭建。捡的书：这一段要诚实、要快，点明从哪捡的就够，把篇幅让给活二和活四。）

### 活四 ｜ f(x)：照出的画面 + 落点

f(x) 有两半：透过这副框照出的画面，加作者落下的那句结论。

画面——戴上这副框，世界看起来是什么样。框架清单型的书，那张清单住在这里（KK 的 12 动词、Kleon 的好偷／坏偷表 + 十条练习）；单概念型的书，绕一个圆心把那幅画面深描。

落点——合上书要你带走的那一句。单独抽出来、脱离上下文，还立不立得住？只是复述书名 = label，不是落点；只是套话（「事物都是辩证的」）= 空话。落点要能立、能反驳、能引用，软话杀掉（「作者认为人性是复杂的」——什么不复杂？）。能直接引作者原话最好，引不到用最贴近的复述——但引语也要过朗读关：英文原话别直译成「X，是 Y 的拙劣替代品」「你是你所有影响的总和」这种系动词加名词的架子，宁可用地道中文意译，也别留译过来的骨头。

### 活五 ｜ reach + 参考系图：这副 f 能照多远

这一件转热：先把整个问题的认知格局画成一张图，再把框推出书外，看它够不够远。

**这一件最容易复发模板病。** 前四件活的形状已经跟着书走了，到这一件，写手最容易把「图 → 走两步 → 预言 → 对账 → 带走那一件」这条流水线连措辞一起搬过去。次序可以铁定，措辞必须随书换。下面教的「承重件、塌、对账、带走那一件」是动作的名字，不是搬进正文的话——上一本书用过的说法，这一本给同一个动作另找一句。两本书的活五并排读，不该是只换了名词的同一段。

**先画参考系图。** 大脑认识任何东西，都是把它挂在参考系的某个位置上（《千脑智能》的底）。各流派对同一个 x 吵了几十年，吵的常常不是事实，是站位——他们站在同一张地图的不同地方，或者用着不同的地图。把所有人摆进同一个参考系，谁挨着谁、谁对着谁、作者从哪挪到哪，一眼看见。delta 在文字里是一句话，在图上是一段看得见的距离。

四步：

1. **提炼轴。** 回到活二的零点：各流派真正吵的是哪个维度？那个维度就是轴，两端各一个极。选哪个维度当轴本身就是立场——作者把哪个维度问都不问，那个就是他不证自明的地基。一个维度用一条数轴就够，别硬凑十字。
2. **钉位置。** 各流派 ●、作者 ★、旧共识文字标明。每个点说得出几何理由：谁离谁近（答案相似）、谁在谁对角（针锋相对）、谁在原点附近（默认立场）。
3. **标 delta。** 旧共识到作者那段距离：往哪个极挪、多远。**作者的位置跟着轴走，不是惯性钉右上。** 捡的书，作者常常就坐在某根共用轴上被重新定位；真换轴的书，作者落在旧轴之外、甚至落在图外——那正是「他的框在旧坐标系里没有位置」看得见的样子。
4. **走两步，验 reach。** 把框推到一个作者从没去过、读者本来也不信的位置，用他的框做一个**可能会错的**预测，再回头对账。

走两步是 reach 的兑现，也是造／捡的证据。框够远——照得到它当初没被造来管的地方，预测还中——这是真造的硬证据。框一推就塌，或者只在原地打转——那是捡，或只在局部成立，照实说。

两条硬规矩：

- **预测要可能会错。** 落在书外一个具体的新位置，给一个能被事实推翻的预言。中了认中，部分中认部分，落空认落空——用你自己的话报，别套一个固定短语。一个诚实的落空，比假装命中有用得多。
- **禁止落回继刚自己的框。** 别把框推到「降秩」「凸性」「四轴」「下行封死上行敞开」上，再说一句「继刚你早就在这么做」。那是奉承回路，不是 reach 测试。走两步要走到书外的世界，不是走回主人的镜子里。

最后，把读者该带走的那一件交出来——就是这副框本身，用这本书的独占标签说出来（「凸性」「偷得好不好」），它的分量已经被 reach 证过了。这一件怎么收，每本书自己找话——别套「合上书十年只记一件」这种固定收尾，那是旧「就在你手里」的换魂，照样是模板。

**图的规矩**（org 输出时）：

- 图包在 `#+begin_example` / `#+end_example` 里（保等宽）。
- 宽度 ≤80 字符。轴有名字，两端有极。
- 流派 ●、作者 ★、旧共识文字标明，三种记号不混。
- 图下给图注：每个点凭什么在那个位置（一点一行）→ delta（含修正／倒转／换轴）→ 末行点作者的轴外之物（他不证自明、看不见的那根轴 = 他的盲点）。
- 没有轴名、没有位置关系、没有 delta 的文字框不是图，是装饰。画不出轴 = 活二／活三没做透，回去重做。

**三种常见几何**（是类型，不是让你抄的范本——按这本书真实的分歧造图）：

- **单轴数轴**：一维分歧。一条横线，两端两个极，各流派钉在线上，作者钉在挪到的新位置，delta 是线上一段距离。
- **十字坐标**：两维分歧。横纵两轴各管一个维度，四象限，各点按两维落位。注意别让作者惯性坐右上——位置由两维决定，不由「谁赢」决定。
- **作者另起一轴**：真换轴。旧流派都挤在旧轴上，作者那根轴跟旧轴垂直、或者干脆画在框外——让「他跳出了原坐标系」肉眼可见。

## 怎么写

冷静、简洁、直白。不夸作者，不贬作者，不替作者辩护，不替作者道歉。拆书人站作者头上看，既不是粉丝也不是反对者。

段内一气呵成。每段说清这一段的核心，展开把它讲透——但**别让所有段长成一个形状**。旧毛病是「开头一句立住、再展开两三句、破折号抖个包袱」，五段下来像节拍器。让形状跟着这本书走：该短的地方一句独占一行，该铺开的地方舒展着写。破折号（——）抖包袱当成口头禅自查，一段最多一个，多了就警报。

短句多。介词砍——「通过／借／凭／在 X 上」能不用就不用，动词直接带宾。

中文白话，汪曾祺、王小波、阿城那一路。每写完一段默念出声，卡的地方整句重写（不是抠字），问一句「汪曾祺会这么写吗」。

不要学术腔。「作者论证了」「作者认为」「X 在第 N 章指出」全部杀掉，直接说作者干了什么。

### 反翻译腔自查（每段写完扫一遍）

英文母语模型用中文最容易犯的毛病。信号词出现 → 整句重写。这是词汇表，不是脚手架。

| 翻译腔 | 真实中文 |
|--------|---------|
| 我接住你 | 我在 |
| 给你拥抱（文字上的） | 抱抱；过来 |
| 你值得更好的 | 你不该这样 |
| 让我们一起... | 我们...（省「让」） |
| 提供支持 | 撑你 |
| 在情感上 X | 情感上 X（省「在」） |
| 这是必要的 | 必须这样；非这样不可 |
| 做出选择 / 做出决定 | 选；决定 |
| 在这个意义上 | 这么说 |
| 拥有...的能力 | 能... |
| 进行（研究/讨论/对话） | 直接用具体动词 |
| 这片领域 | 这门学问；这套东西 |
| 通过 X 抓到 Y | X 动词直接带宾 |
| 一是...二是...三是... | 散文式排开，不编号 |

- **句式英文化**：被动句滥用（「被这事困扰」→「这事缠着」）；「是...的」句（「这是重要的」→「这事要紧」）；长定语拆两句；括号补充整段删；三层定语「X 的 Y 的 Z」拆开（一个「的」没事，两个起停下来）。
- **词性英文化**：动名词化（「做出决定」→「决定」）；抽象集合名词（「领域／阶段／过程／方式」→ 换具体）；形容词副词堆砌（「深刻的、有意义的、重要的」→ 选一个）。
- **意合三省**：连词省（「因为...所以」→ 直接说后半句）；主语省（上下文能补的别重复）；「的」省（「温柔的支持」→「温柔地撑」）。
- **朗读关**：信号词 grep 抓不到的 frame 级翻译腔，靠默念出声捕。每段默念一遍，卡的地方整句重写。

### 禁切痕风

交付的散文里，别用「这一刀」「锋利」（当元描述）「钉死」「砸实」这类自我修辞，预告「下面有重话」。内容够不够力，读者自己看得出，不用作者吆喝。要热，靠把事讲透，不靠预告。

## 一道关（交稿前过一遍，不在正文里写）

旧版每段一个仪式、文末又三道测试，这本身就是模板感的来源。现在合成一道关，交稿前默问，正文里一个字都不留。任何一条过不了，回去重写那一段。

1. **x 读得出吗？** 读者能一口气说出「作者在答什么问题 + 它冲着哪个默认答案」？这话拿去套同领域另一本书，套得上 → 太泛，重写。
2. **f 读得出、裁得清吗？** 这副框用了这本书的独占标签讲（至少一个签名术语／列表／区分）？造还是捡，明判了？hard-to-vary 真跑了（抽一根承重件看塌不塌）？
3. **f(x) 立得住吗？** 落点单独抽出来锋利、能引、能反驳？软话、复述书名、空套话，三样都没有？
4. **reach 诚实吗？** 走两步落在书外一个具体新位置，预测可能会错，没落回继刚自己的框（降秩／凸性／四轴）？中／半中／落空照实报了？
5. **散文过关吗？** 反翻译腔朗读关过了；禁切痕风守了；下面这些词一个都没出现——

   **逐字禁用清单**（旧模板的指纹，出现即重写）：「前五段把骨架抽出来摆着，冷的」「……的精神内核就在你手里」「图能预测书外的事，它就成了你的」「他冲的默认答案」「撑不住了」（作失效诊断的口头禅）「delta 一句话」「档位：」「翻成人话」「意思是：」「为什么是这一件」「走两步」（作小标题）「骨架／抽骨架」（作主导隐喻）「一句话」（作节奏填料）「才动笔」「动笔前」。破折号抖包袱一段最多一个。

   **新模板指纹**（活五尾巴最容易长出来的新模子，跨书逐字复用即重写——禁的是逐字句架跨书复用，不是禁概念）：「对个账」「对账，半中」（作固定收束）、「带走的那一件／该带走的就这一件」（旧「就在你手里」的换皮）、「从仓库里捡／拿」（造／捡裁决的脚手架词，别进正文）、「抽掉……承重件……就塌／散架」（hard-to-vary 的逐字句架）、「把 X 推到 Y，看它照多远」（活五节标题模子）、「这副框剥到底就一句／一个字就装得下」（f 段开场模子）、「合上书……十年……只记一件」（带走那一件的固定收尾，旧「就在你手里」换魂）。这些动作每本书都得做——验塌不塌、诚实对账、带走那一件——但每本得换自己的话说，不许两本逐字一样。

## 怎么找材料

只给书名——
- 先 WebSearch / WebFetch 找作者、导言、目录、核心论点的二手解读。
- 再查这个问题的认知史：作者之前主要流派怎么答（零点的材料）。搜「问题 + history of / schools of / debate / criticism」。
- 拿到核心 argument 和零点再动笔。不要凭印象瞎拆——印象拆出来是 wikipedia 风的空话；零点更不能编，编出来的流派过不了关。

给了 PDF / arxiv / 内容——直接读，不用查。

经典书——可以靠内部知识，但每写一条心里默问：这是书里实际有的，还是我编的？编的删掉。

## 输出

1. 取时间戳：`date +%Y%m%dT%H%M%S` 和 `date "+%Y-%m-%d %a %H:%M"`。
2. 写入 `~/Documents/notes/{时间戳}--拆书-{书名}__book.org`。
3. 输出 org 文件：标题用 `*`，加粗用 `*bold*`（单星号），不用 markdown 的 `**`。
4. 文件头（不变）：

```
#+TITLE: 拆书：《{书名}》
#+SUBTITLE: {作者} | {一句话核心观点}
#+DATE: [{YYYY-MM-DD Day HH:MM}]
#+FILETAGS: :book:{书的领域 e.g. philosophy/biology/economics}:
#+IDENTIFIER: {YYYYMMDDTHHMMSS}
```

5. 文件头之后、第一个 `*` 标题之前，给三行罗盘（x / f（造/捡）/ f(x)）。
6. 正文 4-6 个内容命名的小节，按五件活的次序走（活二薄书可并进活一；活三造的书可拆两段）。活五的参考系图包在 `#+begin_example` / `#+end_example` 里。
7. 报告文件路径给用户。

## 红线（每条必过）

1. **罗盘必给** — 三行 x / f / f(x) 摆在正文之前；f 行带造／捡／捡+加工一词裁决；槽位固定，填充全是这本书的，无套话。
2. **脊柱是 x→f→f(x)，不是六个固定标题** — 标题从这本书长出来，造的书和捡的书拆法不一样；旧的「一核心问题／二共识基线……六认知地图」六连号，一个都不许复活。
3. **造／捡必裁** — f 段下中心裁决，先验默认捡，hard-to-vary 真跑过才升造；别到处判造。
4. **零点必摆** — 2-4 个已在场的 f 各一句，标出共识，正对作者不满；缺零点 = delta 没零点。
5. **delta 照实说，不排名次** — 修正／倒转／换轴据实判，禁「换轴最大」的诱导，禁默认往换轴靠；多数书是修正或捡。
6. **独占标签必含** — 至少一个只有这本书才有的术语／列表／区分；少了它，拆的是领域不是这本书。
7. **f(x) 落点不能软** — 能立、能引、能反驳；软话、复述书名、空套话杀掉。
8. **走两步落在书外，预测可能会错** — 具体新位置 + 可证伪预测 + 诚实对账（中／半中／落空都认）；**禁落回继刚自己的框**。
9. **参考系图必真画** — 提炼轴（各流派真分歧的维度，图注点作者轴外之物）+ 钉位置（● 流派 / ★ 作者 / 旧共识，几何理由说得出，作者位置跟轴走不惯性钉右上）+ 标 delta（图上可见的距离）+ 走两步；没有轴名／位置／delta 的装饰框不算图。
10. **反翻译腔 + 禁切痕风** — 自查表全过，朗读关默念汪曾祺会不会这么写；逐字禁用清单清空；金属比喻（这一刀／锋利／钉死／砸实）一个不许。
11. **不夸不贬不替辩护** — 站作者头上看，不是粉丝也不是反对者。
12. **一道关，不是每段一个仪式** — 交稿前过五条关，正文里不留测试痕迹。
