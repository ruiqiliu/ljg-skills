# Reading Guide：从一本书里带走一个变化

正文不是全书摘要。它只解释一件事：这本书为什么会让读者在某个具体时刻，多看见一点、换一个问题、换一个动作，或克制一个旧反应。

先在内部完成选择，再开始写。不要边填章节边找主线。

## 1. 先看材料

只给书名时，至少查：

- 作者与出版社页面。
- 目录、导言、样章或可核实的书内片段。
- 作者访谈或可靠书评。
- 这个问题之前常见的回答。

给了 PDF、正文、样章或旧笔记时，先读原文。除非用户禁止联网，仍轻查外部基线，避免只接受作者自己树立的对手。

按材料决定说到什么程度：

- *完整拆书*：有全文或足够书内材料。
- *初拆*：只有目录、导言、样章、访谈或可靠书评。
- *假设版*：资料很薄，只能提出有标记的暂定理解。

材料等级放在末尾「资料校准」，不要在开头打断阅读。

## 2. 找到真实问题

问题不是话题。

「科技与未来」是话题。「科技演化有没有自己的方向」才是问题。

问：

- 作者为什么非写这本书不可？
- 读者原来通常怎么理解或应对？
- 哪个具体困境让旧答案失灵？
- 如果全书只回答一个问题，那句话是什么？

拿去描述同领域另一本书也成立，就还太泛。

## 3. 先选变化，再写摘要

在内部找出两三个候选变化，不要把候选全部写进成稿。

候选可以是：

- *看见*：让读者发现一个过去忽略的信号。
- *提问*：把旧问题换成一个更有用的问题。
- *行动*：替换一个小动作或检查顺序。
- *克制*：在旧反应发生前多停一下。

只选同时通过四个检查的那个：

1. *书的指纹*：能由书中独有的场景、实验、人物、区分或比喻支撑。
2. *前后不同*：能说清读者原来会怎样，之后会怎样。
3. *触发清楚*：能指出它会在什么具体情境出现。
4. *足够小*：不必记住整套框架，下一次就能使用或察觉。

最中心的命题不一定胜出。宁可带走一处小变化，也不要带走一套永远想不起来的体系。

## 4. 给变化找一个记忆锚

只选一个具体载体：

- 一个故事或人物片段。
- 一个实验或反常结果。
- 一个反复出现的意象或比喻。
- 一个作者自己的区分。
- 一个让旧答案失灵的案例。

正文始终围绕这个锚解释。列出很多术语不能代替一个能想起来的画面。

必须出现 1 到 3 个书中自己的词，但术语只负责证明这是这本书；记忆锚才负责让收获留下来。

## 5. 把它落到读者身上

上下文里若有读者正在处理的真实问题，可以用它选择触发情境。没有这些信息时，不猜职业、关系、情绪或人生阶段，只写普通人能自行识别的可观察场景。

变化句优先写成：

> 下次当 X 出现，我先 Z，而不是 Y。

不同书型可以自然变体：

- 方法和论证类书，通常落到一个新问题或小动作。
- 历史和传记，通常落到一个判断信号或因果检查。
- 小说、诗歌和回忆录，可以落到注意、理解、容纳或克制，不必硬写行动步骤。

如果一句话里出现多个「先」、一串问号或三项以上动作，说明收获又长成了清单，继续缩。

## 6. 写四段正文

### 它在解决什么

从一个具体困惑、冲突或失败开场。写清读者原来的反应，以及它为什么在这里不够。

### 它让我看见什么

用一个书中记忆锚解释作者的洞见。只补理解这件事所需的最少机制，不顺着目录展开。

### 下一次，哪里会不同

先写变化句，再用一小段解释为什么这个替代反应更好。

只有存在真实误用风险时，紧接着补一句限制。不要另设「边界」章节，也不要把这一段写成应用清单。

### 资料校准

列 3 到 6 条真正支撑主线的材料：

- 书内证据：全文、样章、目录或导言。
- 作者材料：访谈、演讲、主页或出版方介绍。
- 外部基线：前人观点、争论或可靠批评。
- 材料等级：完整拆书、初拆或假设版。

资料只服务所选变化，不要扩成领域综述。

## 7. 输出格式

保存到：

`~/Documents/notes/{时间戳}--拆书-{书名}__book.org`

时间戳：

- `date +%Y%m%dT%H%M%S`
- `date "+%Y-%m-%d %a %H:%M"`

文件头：

```org
#+TITLE: 拆书：《{书名}》
#+SUBTITLE: {作者} | {一句话核心变化}
#+DATE: [{YYYY-MM-DD Day HH:MM}]
#+FILETAGS: :book:{领域}:
#+IDENTIFIER: {YYYYMMDDTHHMMSS}
```

## 8. 文风

简洁、直白，像人说话。不要靠形容词制造力量。

少用「作者认为」「第 N 章指出」「在这个意义上」「提供了一种可能性」。多直接写「他把 X 改成 Y」「这个场景说明了什么」「下次可以先做什么」。

禁用旧口头禅：骨架抽出来、精神内核就在你手里、走两步、delta、钉死、这一刀、锋利、砸实、带走的那一件。

## 9. 交稿前最后一问

补完这句话：

> 因为读了这本书，下次在 ______ 时，我会 ______。

填不出来，或者需要填四项以上，说明还没有拆到位。
